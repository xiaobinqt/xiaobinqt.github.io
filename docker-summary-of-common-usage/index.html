root 用户执行

有时进入容器后，用户就是变成非 root 用户，这种时候又没有密码，在执行一些操作的时候就会非常不方便，这是可以用 -u root 来指定用户。

执行简单命令可以这样👇

如果需要进入容器，可以这样👇

1
2
3

docker exec  -u root -it 容器名  bash
#或者
docker exec  -u root -it 容器名  sh
