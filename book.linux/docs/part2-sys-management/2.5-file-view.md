# 2.5 文件查看命令

## 2.5.1 cat

参看文件打印到标准输出。

## 2.5.1 head

查看文件开头，比如前 5 行`head -5 文件名`。

## 2.5.1 tail

查看文件结尾，查看最后 3 行`tail -3 文件名`。

可以使用`-f`跟踪文件变化。

## 2.5.1 wc

统计文件的字节数、字数、行数。

### -l 参数

统计行数。
