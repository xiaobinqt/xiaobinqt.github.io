# 4.3 数组

+ 定义数组

```shell
IPTS=(10.0.0.1 10.0.0.2 10.0.0.3)
```

+ 显示数组的所有元素

```shell
echo ${IPTS[@]}

```

+ 显示数组元素个数

```shell
echo ${#PTS[@]}
```

+ 显示数组的第一个元素

```shell
echo $IPTS[0])
```
