    4.8 函数 #

    函数用于 “包含” 重复使用的命令集合。

    4.8.1 自定义函数 # 

    function fname(
  # 命令
}

# 函数的执行 `fname`

    function 关键字可以省略。

    变量和参数 #

    函数作用范围的变量可以使用 local 申明，只在函数内部生效：

    local 变量名

    函数的参数：$1 $2 $3 ${10} .. $n

    4.8.2 系统脚本 #

    系统自建了函数库，可以在脚本中引用，目录是 /etc/init.d/functions，如果想导入系统函数库，可以使用 source 函数脚本文件 “导入” 函数。
