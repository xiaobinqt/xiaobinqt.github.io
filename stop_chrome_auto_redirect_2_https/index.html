How to Stop Chrome from Automatically Redirecting to https

xiaobinqt 收录于 开发者手册

目录

这几天在使用 google 浏览器打开公司的一个网站时，发现总是自动跳转到 https，以至于出现下面这个页面：

有时候浏览器太智能了也不是一件好事🤣。

解决方法

用浏览器打开chrome://net-internals/#hsts 这个页面，在最下面的 Delete domain security policies 填上需要禁止跳转的网站，然后点击Delete。

这里有个需要注意的地方是，如果我们的网址是 http://g.xiaobinqt.cn:8000，那么就需要在这里填上 xiaobinqt.cn。
