## 简介

+ Python 是一种高级编程语言
+ Python 是解释型语言
+ Python 语法和自然语言很像
+ 足够流行，应用的行业广泛
+ 有着丰富的库（标准库和第三方库）

### 不适用场景

+ 抢购、电商类计算密集型场景
+ 涉及到内存等底层硬件操作
+ 网页、小程序等前端开发
+ App 开发
